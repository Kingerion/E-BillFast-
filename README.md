# EbillFast Official Website

Static website for **e-billfast.gr** ready for GitHub Pages.

## Files
- `index.html` – landing page
- `privacy.html` – privacy policy
- `terms.html` – terms of use
- `styles.css` – styling
- `CNAME` – custom domain

## GitHub Pages
1. Upload all files to your public repo
2. Go to Settings → Pages
3. Source: main branch / root
4. Custom domain: `e-billfast.gr`
5. Make sure your DNS has the GitHub Pages A records
